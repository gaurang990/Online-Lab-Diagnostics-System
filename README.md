# 🏥 Diagnostic Lab Reporting System

A complete **Diagnostic Lab Management and Reporting System** built with the **Django Web Framework (v2.2 LTS)**.

This system helps diagnostic centers manage patients, laboratory tests, reports, staff, and administrative operations efficiently.

---

## ✨ Features

- 🔐 User Authentication System
- 👨‍💼 Diagnostic Admin Dashboard
- 👨‍⚕️ Staff Management Dashboard
- 🧪 Laboratory Test Management
- 📋 Patient Report Management
- 📊 Order Tracking System
- 🖨️ Report Generation
- 🛡️ Secure Admin Panel
- 📱 Responsive User Interface

---

## 🛠️ Technology Stack

| Technology | Version |
|------------|---------|
| Python | 3.x |
| Django | 2.2 LTS |
| Database | SQLite |
| Frontend | HTML, CSS, JavaScript |
| Backend | Django Framework |
